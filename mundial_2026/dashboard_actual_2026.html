Qué significan las métricas

Marcador proyectado muestra el resultado entero mas probable. Promedio estimado de goles del modelo es una media probabilistica y por eso puede llevar decimales. Probabilidades de resultado es la chance de victoria, empate o derrota en ese mismo corte. Puntos FIFA entran como señal estructural secundaria, con menos peso que Elo. Coincidencia entre modelos ayuda a separar picks claros de partidos realmente parejos.

El dashboard pasa a modo in-play: usa minuto, marcador actual y fase del juego para recalcular el resultado final más probable, el resto de goles esperados y los marcadores finales mas probables en cada corte de actualizacion.

Cómo Leer Esto

Como esta armado el modelo

La idea aqui no es complicar por complicar. El modelo ya es fuerte para quiniela; lo que mas valor agrega ahora es que explique bien por que cambia y que luego podamos medir con datos reales si viene acertando.

Partido a partido

Combina fuerza de cada seleccion, contexto del partido y un stack de modelos para estimar marcador final y probabilidades de victoria, empate y derrota.

Cuadro completo

La llave publicada se construye con Monte Carlo dinamico de 15.000 iteraciones por corrida para que el cuadro no cambie solo por ruido de simulacion.

Stack estadistico

La capa prepartido mezcla el modelo principal Bivariante Poisson, un modelo de contraste Poisson independiente, un ajuste de baja anotacion y un ensamble ligero final. Si las cuotas son confiables, tambien se usan como referencia suave y no como sustituto del modelo.

Estado dinámico

Actualiza Elo, forma, fatiga, disponibilidad, disciplina, clima, alineaciones, bajas y mercado a medida que aparecen datos nuevos. Ademas, acumula el estilo reciente de cada seleccion para no arrancar cada cruce desde cero.

Modo in-play

Durante un partido, condiciona las probabilidades por minuto, marcador actual y fase del juego. Si el feed trae datos mas ricos, tambien usa tiros, tiros al arco, posesion, calidad de ocasiones, corners, disciplina y expulsiones para recalcular todo.

Noticias y bajas

Si el feed publica ausencias, cambios de XI o noticias relevantes del partido, esas señales entran como disponibilidad, moral o contexto adicional.

Como validar el refresh

La portada publica hora de actualizacion, badge En vivo, minuto modelado y un latest.json. Si esos campos cambian, el in-play se esta recalculando bien.
